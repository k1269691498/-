My Module

This is the optional UI page for your module.

Return _openModule: true from Java to open this page from chat. Use it for settings, status, previews, or any interactive workflow that does not fit in a chat bubble.

Starter actions

  • hello returns a simple JSON response and HTML mini-card.
  • doTask demonstrates parameter validation and structured output.
  • savePreference and readPreference use module-private storage.
  • renderSummary demonstrates table output via _displayHtml.
